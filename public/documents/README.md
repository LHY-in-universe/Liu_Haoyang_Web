# 📄 文档管理系统

本目录用于管理网站中的各类文档资源。

## 📁 目录结构

```
documents/
├── resume/          # 简历文件
│   ├── main.pdf     # 主简历
│   └── ...
├── presentations/   # 演讲稿和PPT
│   ├── PHYS/       # 物理相关演讲
│   └── ...
├── academic/        # 学术论文
│   ├── PHYS/       # 物理论文
│   └── ...
└── blog/           # 博客文章备份
    └── ...
```

## 🔧 文档管理

### 添加新文档
1. 将文档放入相应目录
2. 在对应的JSON配置文件中添加元数据
3. 运行构建脚本更新页面

### 支持的格式
- PDF文档
- Markdown文章
- PowerPoint演示文稿

## 📝 元数据格式

每个文档都应该包含以下信息：
- 标题
- 作者
- 创建日期
- 描述
- 标签
- 文件大小
- 下载链接
