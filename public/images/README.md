# 📸 图片资源目录

本目录存放网站的所有图片资源。详细的图片管理说明请查看: [图片管理文档](../docs/IMAGE-MANAGEMENT.md)

## 📁 目录结构

```
images/
├── posts/          # 博客文章图片
├── articles/       # 文章封面和缩略图
└── common/         # 通用图片资源
```

## 🚀 快速使用

```bash
# 扫描图片使用情况
npm run image-scan

# 优化图片
npm run image-optimize

# 创建文章图片目录
npm run image-create article-slug

# 清理未使用图片
npm run image-cleanup
```

更多详细信息请查看 [图片管理文档](../docs/IMAGE-MANAGEMENT.md)。
